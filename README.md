# SafePatrol 管理工具

这是一个基于 Electron 的桌面应用，用于读取 SQLite 数据库并进行点检记录查询与导出。

## 功能

- 左侧为暗色风格菜单栏，选中项会高亮显示。当前只有“点检详情与导出”可用，其它入口（漏检率统计、点检项管理、员工管理）作为占位。
- 顶部工具栏采用扁平按钮样式，包含数据库选择、路线选择、日期输入、预览与导出。
- 界面使用现代 CSS 变量和简洁配色，表格为滚动视图，列宽可拖动调整，字号缩小以增强信息密度。

1. 选择 SQLite 数据库文件
2. 根据数据库中的 `routes` 列表选择路线
3. 选择日期
4. 预览当日点检明细，结果在表格中展示
5. 导出包含当日所有点位的详细检查信息的 Excel 文件。导出时应用会重新查询数据库生成“日期 时间 路线名 点检人 值 班次 点位ID 点位名 设备名 点检项 点检频率 点检频次 检测值 是否正常”格式的表格。

* 如果某一时段点位已有检查，将输出每条检查项目的完整信息；
* 如果某一时段点位未检查，则只生成一行，时间、点检人、设备名、点检项和检测值留空，在“是否正常”列中以红色字体显示“未检”。

预览仅为了查看总体情况，导出结果不再与预览表格完全一致。
- `值` 列根据五天一周期的排班规则自动计算。周期定义：甲乙丙丁戊五个值按照 `白班 → 夜班 → 休息1 → 中班 → 休息2` 的顺序循环。已知 2025-12-05 是戊值夜班、甲值白班、丙值中班、丁值休1、乙值休2，程序从这个基准计算任何日期的值。查询结果中的 `值` 列会显示记录当时对应的当前值。
